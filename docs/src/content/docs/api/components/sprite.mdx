Sprites are the base for all textured objects that are rendered to the screen

## Usage

```svelte live
<script>
  import * as PIXI from 'pixi.js'
  import { Sprite } from 'svelte-pixi'
</script>

<Sprite
  texture={PIXI.Texture.from('/assets/adventurer.png')}
  anchor={0.5}
  scale={3}
/>
```
